# Hello there

This is some basic syntax for the language.

## Variables

Variables are declared with the `let` keyword.

```ab
let x = 42
```

### Variables can be reassigned

```ab
main {
    let x = 42
    x = 38
    x = [1.2, 323, 3]
    x = 'Hello World'
    'This is an interpolation {24 + 12 + 'nested {456} interpolation!'}'
    $mkdir {name} test$?
    read()
    read(a + 12)
}
```
