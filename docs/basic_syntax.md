## Variables

Variables are declared with the `var` keyword.

```ab
let x = 42
```

### Variables can be reassigned

```ab
main {
    let x = 42
    x = 38
    x = [1.2, 323, 3]
    x = 'Hello World'
    $mkdir test$?
}
```
